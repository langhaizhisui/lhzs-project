iPhone8与iPhoneX选哪个 苹果8和苹果X哪个好 iPhone8与iPhoneX对比
栏目： 选购导购,数码科技 作者： admin 时间： 2017/09/15

北京时间9月13日凌晨一点，苹果发布会在美国开幕，与预测不同，iPhone 8外观和尺寸都并无太大变化，铝合金机身，并用上了玻璃材质。iPhone 8 和 iPhone 8 Plus 支持防水，支持 3D Touch 和双扬声器技术。

而“one more thing”环节中的iPhone X却抢足了戏，iPhone X搭载了 5.8 英寸全屏幕设计，前后均为玻璃设计，机身防尘防水。屏幕 ppi 高达 458，材质为 OLED，支持 Dolby Vision 和 HDR10，无home键，竖排双摄像头、引入face ID脸部识别，更有新奇的Animoji。

此次苹果发布会盛况，不乏看到中国的身影，天猫作为国内首发平台受邀至苹果发布会现场，并在天猫和优酷上为国内带来高清的现场直播。

这次发布会是自乔布斯发布iPhone4后的第十年，乔布斯发明了iPhone可以说是重新定义了手机，而十年后库克给我们带来了什么——iPhone8和iPhoneX。这个时候就有不少人纠结买iPhone8还是买iPhoneX？接下来详细介绍一下：

iPhone8与iPhoneX该选哪一个？——显示屏

iPhoneX是5.8寸，iPhone8P是5.5寸，iPhone8是4.7寸。

iPhone8与iPhoneX该选哪一个？——摄像头

均采用前置800万后置1200万，不同的是iPhoneX和iPhone8P采用了双摄像头且具有人像模式。

iPhone8与iPhoneX该选哪一个？——续航

均采用无线快速充电，iPhone X比iPhone7使用时间长2小时，其他两款均与上代基本相同。

iPhone8与iPhoneX该选哪一个？——其他

iPhoneX采用人脸识别解锁，iPhone8和iPhone8P依然采用指纹识别。芯片均采用自家A11处理器，具有仿生AR功能。

　iPhone8与iPhoneX该选哪一个？——售价

　　　　苹果iPhoneX比iPhone8好在哪，很多朋友看完发布会仍然蒙蒙的，苹果官网一张图，就可以看到两个产品的差距。都拿64GB举例，iPhone8售价5888元起，iPhone8 Plus售价6688元起，iPhoneX售价8388元起。9月15日正式开始预售。
