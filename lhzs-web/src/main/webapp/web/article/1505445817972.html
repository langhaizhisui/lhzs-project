1.       硬件设施：电脑一台，笔记本电脑、台式电脑均可，并可以上互联网。对电脑配置没有要求。当然，电脑配置越高、操作越快，这个不用多解释。

2.       自身必须年满16周岁，并且有二代身份证，在工商、建设、或者农业银行（推荐这三个银行是因为这三个银行开网店的操作程序最简单、好操作、安全系数较高）办理一张银行储蓄卡，也就是我们平时说的“银行卡”（非信用卡，信用卡不能开店）并且绑定现用的手机号码。

3.       属于自己的一部手机并能够正常通话状态（是为了你注册淘宝、安装一些淘宝必要的辅助组件时作为接收验证码来用）
